# Crafty Carts (REMAKE!)
Crafting stations on wheels

Client side only, no server install required.

This is a remake of the original mod by RoloPogo (all credit for the assets and 90% of the code goes to them). Removed the dependency on ANY Lib.

## Features
- Workbench, Forge and StoneCutter carts
- Craft and repair at carts
- Build near carts like stations
