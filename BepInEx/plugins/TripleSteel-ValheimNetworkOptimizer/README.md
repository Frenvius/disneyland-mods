What it does:
Increases the data transfer cap (Yes this works post patch 0.148.6)
Fixes the buffer limit error some servers owners would see when players would pass a certain data per second limit.


Who should install it:
This modification works best if both the server itself and all of the clients connecting to the server install this mod. 

In some instances where the dedicated server may be running some other mods that modify data transfer speed this mod is not needed but will still help if players joining that server install this mod. 

Update as of Valheim 0.148.6:
I have not seen any other mod altering buffer speed on servers properly so be careful XD. Let me know if you see one so I can confirm it works ok with this.

Do I really need this mod post Valheim patch 0.148.6
Honestly that entirely depends on you! If you're still experiencing de-sync issues this can help.

If your playing with 2-3 people casually this probably wont help that much because the new vanilla network fixes they released are actually not bad. If you are playing with more than that or building very large bases you may want to consider giving this mod a try. 


What it doesn't do:
Anything else; this is on purpose. This mod is intended to be as stackable and simple as possible so you can use it freely while maintaining the type of gameplay you want.
