## Description
PotionsPlus formally Odin's Potions Plus brings you several new adventuring potions to help you stay alive against many of the dangers ahead.
The potions are scaled in a crafting order to fit right in with the progression of valheim, starting with crafting your Alchemy Table you will be on your way to unlocking the art of potion crafting. Potions will require various materials that you unlock as your progress through Valheim. The new Cauldron allows us to craft the advanced potions known as Flasks. By breaking down a "Greater Potion" we can create a PotionBase Mead that is used to craft our end tier Flasks.


## Installation
Install with Thunderstore Mod Manager
### Manual Install
Install Jotunn JVL per mod author's insturctions.
Directory structure should look like this:
```
BepInEx ->
    plugins ->
        PotionsPlus 2.0 -> DLL and Assets Folder inside
```
